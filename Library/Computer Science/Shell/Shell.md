## Qué es la shell?

**Shell** es el término del ecosistema [UNIX](UNIX.md) para la interfaz de usuario de terminal del sistema, que permite comunicarte con la computadora por medio del teclado y el monitor.
Una **shell** como concepto es simplemente un programa que interpreta comandos de texto para realizar las acciones del [[Sistema Operativo (OS)|sistema operativo]]. Similar a cualquier programa o interfaz gráfica, las **shells** usan las [[|system calls]] para realizar cualquier acción.

Piensa a la **shell** como una "navaja suiza" o como un "lanzador de comandos". Es un programa independiente al sistema operativo, que su única tarea es interpretar los comandos en lenguaje shell que le enviamos.

Algunas de las shells mas utilizadas son; La '[[Bourne Again shell (bash)]]', una shell moderna de propósito general. La '[[Korn shell (ksh)]]', y la '[[Tenex C shell (tcsh)]]'.
