#shell #unix

**Shell** es el término [UNIX](UNIX.md) para la interfaz de usuario de terminal del sistema, que permite comunicarte con la computadora por medio del teclado y el monitor.
Las **shells** son simplemente programas que encapsulan todas las acciones del [sistema operativo](Sistema Operativo (OS), y existe una gran variedad para elegir.

Algunas de las shells mas utilizadas son; La '[[Bourne Again shell (bash)]]', una shell moderna de propósito general. La '[[Korn shell (ksh)]]', y la '[[Tenex C shell (tcsh)]]'.
